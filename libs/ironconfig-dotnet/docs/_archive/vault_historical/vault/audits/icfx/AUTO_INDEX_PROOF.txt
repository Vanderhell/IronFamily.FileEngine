9. CRITICAL DESIGN DECISIONS
============================

âś“ Threshold = 13:
  - Chosen based on cost/benefit analysis
  - Below threshold: linear scan faster
  - At threshold: hash table overhead justified

âś“ Per-object decision:
  - Allows optimal encoding for mixed-size objects
  - Root array can contain both 0x40 and 0x41
  - Decoder handles transparently

âś“ Deterministic based on pairCount only:
  - No random selection
  - No encoder state beyond pairCount
  - Same input guaranteed same output

âś“ Mode as enum instead of bool:
  - Explicit over implicit
  - Easier to extend in future
  - Clearer intent in code
