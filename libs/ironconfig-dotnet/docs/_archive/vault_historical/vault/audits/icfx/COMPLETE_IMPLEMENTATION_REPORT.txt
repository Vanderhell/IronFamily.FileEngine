Mode: --index auto

CRITICAL DESIGN DECISIONS
==========================

1. Threshold = 13 (JUSTIFIED)
   - Based on cost/benefit analysis
   - Below threshold: linear scan faster
   - At threshold: hash table overhead justified
   - Sweet spot for typical objects

2. Per-Object Decision (JUSTIFIED)
   - Allows mixed 0x40/0x41 in same file
   - Reader already supports both transparently
   - Optimizes each object individually
   - No global mode lock-in

3. Mode as Enum (NOT BOOL)
   - More explicit than boolean
   - Easier to extend in future (add more modes)
   - Clearer intent in code
   - Reduces confusion (what does true mean?)

4. Deterministic Based on Field Count Only (JUSTIFIED)
   - No random seeds
   - No environment dependencies
   - Reproducible across machines/runs
   - Safe for cryptographic validation
